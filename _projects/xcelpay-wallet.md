## Context

XcelPay Wallet is a crypto wallet platform built to make day-to-day cryptocurrency usage more practical. The product combines wallet management, token swaps, and payment-oriented utilities like gift cards and mobile top-ups, helping users move from passive holding to active use.

![Project Image](/assets/img/project/xcelpay/xcelpay-mutii.webp)

## My Role

I contributed to product delivery across key wallet capabilities focused on utility, accessibility, and transaction confidence.

My contributions included:

- supporting buy, sell, and swap workflows across supported crypto pairs
- contributing to wallet experiences for managing multiple assets in one app
- enabling utility features such as gift card purchases and mobile top-ups
- helping maintain approachable UX for users with varied crypto experience levels
- supporting production feature delivery across mobile wallet surfaces
