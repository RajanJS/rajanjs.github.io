## Context

Hamro Blood Bank is a mission-driven platform built to reduce blood shortage emergencies in Nepal. The core problem is time-critical donor discovery, where delays directly affect patient outcomes. The product supports faster matching between requests and available donors.

![Project Image](/assets/img/project/hamroblookbank/hamro-blood-bank2024-07-09-200558.webp)

## My Role

I helped build and deliver a specialized platform for Helping Blood Search to streamline donor operations and request handling.

My contributions included:

- implementing donor profile and availability workflows
- building request intake and matching-oriented application flows
- enabling alerting and reminder behavior for donors and recipients
- supporting secure storage and privacy-aware handling of sensitive records

## Technical Challenges

- handling urgent, real-time matching requirements in a high-stakes domain
- keeping donor availability data current enough for practical dispatch decisions
- balancing fast workflows with data security and privacy constraints
- keeping operational workflows simple for non-technical organizations
